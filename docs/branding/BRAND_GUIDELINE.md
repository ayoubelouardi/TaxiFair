## 4. UI/UX Principles (The "Static" Look)

Since Dalil has no database, the UI should feel **instant**.
1.  **Geometric Patterns:** Use subtle, low-opacity Islamic geometric patterns (Girih) as background watermarks.
2.  **Skeuomorphic Touches:** Use shadows that mimic the desert sun (sharp, high-contrast).
3.  **RTL First:** The layout must be optimized for Right-to-Left flow, ensuring the Arabic script feels native, not "translated."

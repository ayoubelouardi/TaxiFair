## Packages
leaflet | Core mapping library
react-leaflet | React components for Leaflet maps
framer-motion | For smooth UI transitions and animations
date-fns | Date formatting for travel times
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely

## Notes
- Leaflet requires CSS to be imported globally or in the component.
- The map will default to Casablanca (33.5731, -7.5898).
- Transport modes and cities are fetched dynamically from the API.
